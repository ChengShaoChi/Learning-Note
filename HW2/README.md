### Heap Sort簡述
* 堆積排序法，利用Heap結構來完成排序。Heap，堆積，是電腦科學中的一種特別的樹狀資料結構，且是Complete Binary Tree（完全二元樹）。可以分為「Min Heap（最小堆積）」與「Max Heap（最大堆積）」兩種。
### Merge Sort簡述
* 合併排序法，是排序演算法的一種，使用Divide and Conquer（分治法）的演算法來實作。
