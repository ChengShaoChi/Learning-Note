## String Definition
* 題目：

Which of the given string definitions are incorrect?
1. s = 'abacaba'
2. s = "abacaba"
3. s = ''abacaba''
4. s = ""abacaba""
5. s = '''abacaba'''
6. s = """abacaba"""

* 意思：哪個選項定義不正確。
* 我的理解與學習筆記：5跟6是三組引號括起來，包含的字串可由多行組成，可表示大段的敘述性字串。

- [x] Definitions 3 and 4 .
