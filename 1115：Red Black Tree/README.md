## Red Black Tree
紅黑樹，是一種自平衡二元搜尋樹。
![image](https://upload.wikimedia.org/wikipedia/commons/thumb/6/66/Red-black_tree_example.svg/675px-Red-black_tree_example.svg.png)
* 特性：
1. 節點是紅色或者黑色。
2. 根節點是黑色。
3. 每個葉子的節點都是黑色的空節點（NULL）。
4. 每個紅色節點的兩個子節點都是黑色的。
5. 從任意節點到其每個葉子的所有路徑都包含相同的黑色節點。
* 操作：
  * 變色
  * 旋轉
    * 左旋轉
    * 右旋轉
